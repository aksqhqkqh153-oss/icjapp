[Apple App Store 등록 절차 요약]
1. Apple Developer Program 가입
2. App Store Connect에서 앱 생성
3. 앱 개인정보(App Privacy) 작성
4. 앱 빌드 업로드(TestFlight 또는 App Store Connect)
5. 앱 리뷰 정보 입력
6. 심사 제출
7. 승인 후 출시

[핵심 심사 포인트]
- 앱 개인정보 고지 정확성
- 로그인 필요 시 리뷰용 계정 제공
- 계정 생성 앱이면 앱 내부 계정 삭제 기능 제공
- 권한 요청 사유가 명확할 것
- 앱이 실제로 동작할 것
- UI가 모바일 앱으로서 사용 가능할 것

[현재 프로젝트 기준 주의]
- iOS 빌드는 Mac + Xcode 필요
- 운영 서버 HTTPS 필요
- 앱 내부 계정 삭제 기능 점검 필요
- 스토어 메타데이터/스크린샷 필요
