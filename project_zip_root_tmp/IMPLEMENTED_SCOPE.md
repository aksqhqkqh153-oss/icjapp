# 구현 범위 점검표

## 직접 구현한 항목
- 회원가입 / 로그인 / 로그아웃
- 로그인 화면 분리, 로그인 성공 후에만 홈 진입
- 복구 이메일 6자리 코드 기반 비밀번호 재설정 (데모 모드)
- 프로필 편집 / 위치 저장
- 홈 피드 / 좋아요 / 북마크 / 댓글
- 사용자 탐색 / 팔로우 / 친구 요청 / 채팅 요청
- 1:1 채팅
- 그룹채팅방 생성 / 입장 / 메시지
- 음성방 생성 API / signaling 저장 API
- 지도용 사용자 좌표 / 지역 GeoJSON 조회 API
- 모임 일정 / 후기
- 게시판
- 캘린더
- 문의
- 신고 / 관리자 신고 종료 처리
- SQLite 샘플 DB / 데모 계정
- Render / Docker 배포 기초 파일
- Windows 실행 BAT 파일

## 부분 구현 / MVP 처리
- 지도 시각화는 Leaflet 완전 연동 대신 좌표 기반 목록형 MVP UI로 제공
- 음성 기능은 TURN 서버까지 포함한 완전 운영형 WebRTC가 아니라 signaling API + room 생성 중심 MVP
- 첨부파일 저장은 외부 스토리지 연동 없이 텍스트/URL 기반 구조
- 프런트엔드 UI는 모바일 우선 단일 테마 MVP
