[추가된 로그인 보안 기능]
- 계정 실패 5회 누적 시 10분 잠금
- 동일 IP 실패 10회 누적 시 10분 잠금
- 실패 10회 이상 누적 시 60분 확장 잠금
- 로그인 성공 시 계정/IP 잠금 자동 초기화
- 관리자용 잠금 조회 API: GET /api/admin/auth-security/locks
- 관리자용 계정 잠금 해제 API: POST /api/admin/auth-security/unlock-user
- 관리자용 IP 잠금 해제 API: POST /api/admin/auth-security/unblock-ip
- 운영 스크립트: python backend/scripts/reset_login_lock.py --login-id wlgns123

[중요]
- wlgns123 잠금 해제는 이 스크립트를 운영 DB 연결 상태에서 1회 실행해야 실제 해제됩니다.
- backend/.env 또는 환경변수 DATABASE_URL이 운영 DB를 가리켜야 합니다.
